Contacts Screen

The list of your contacts from which to pick an address.

Hint: you can choose to sort the contacts by first name or by surname in the app settings.

Search on Contact Name

You can use the search bar to search for a contact name.

Picking a contact address

Just find and select the contact from the list that has the address you are after.

If the contact has multiple addresses (e.g. home, work, etc), then a dialog will pop up to allow you to select which of the addresses you want to use.
