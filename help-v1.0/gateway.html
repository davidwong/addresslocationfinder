Use Location Screen

This screen will show the details of the location of the address, and a list of things you can do with it.

Note: the list of apps and websites available to use the location with will vary, depending on the version you have installed.

App & Web Examples

These are some of the applications and websites that you can use with the address location.

An App is Greyed Out

If an app in the list of apps/websites is greyed out, this means it is disabled because that application is not currently available.

Some reasons that an app may be unavailable to use include:

  • The app is not installed on your device.
  • The app is not supported for this version of Address Location Finder.

Apps with Navigation / Directions

Some apps have a mode for navigation or directions with routing.

Apps that have a navigation mode will have a navigation icon on the right.

When you select an app with a navigation icon, it will use the location as a destination for navigation or directions if the app setting for navigation mode has been enabled. Otherwise the app will use the location in the standard way.

An icon at the top will show if the navigation app setting is on

or off ...

How to Use Websites

If an item in the list is a websites, then when you select it, this is what happens ...

1. The website selected in loaded into a browser window.

2. The coordinates (latitude and longitude) of the address location you searched for is inserted into the appropriate location (i.e. form field) in the webpage that is loaded.

3. You can then fill in whichever other form options that you require on the web page to do what you want with the location.

Website Not Working?

If you select a website from the list, and it doesn't seem to be working, here are some things to check.

- If the website fails to load, check that you still have a network connection to the internet. It may also be possible that the website is down (e.g. for maintenance).

- The website is loaded into an internal browser for the app, not your normal web browser. So you can only use it for the web page that you selected, and you cannot do normal browser functions like refresh or go back.

- You need to wait until the web page has finished loaded completely before using it.

Hint: Check the website loading bar at the top of the browser screen to see when the page has finished loading.
