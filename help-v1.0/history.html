History Screen

The list of history snapshots of location searches. Clicking on a history snapshot will reload that location again.

Hint: History searches are saved automatically upon a successful search.

History menu

Tapping the menu icon for a history snapshot will bring up a menu of the actions that can be performed on that history item.

History menu actions

The user configurable application preferences.

View Details - Show the details for the history snapshot, including the original search parameters and the location.

Add to Favourites - You can save the history snapshot to your list of favourite locations.

Delete - Delete the selected history snapshot.

Delete all History - Delete all snapshots to clear the history.
