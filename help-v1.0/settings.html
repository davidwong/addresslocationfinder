App Settings

The user configurable application preferences.

Note: the preferences available may vary, depending on the version you have installed.

Contact Names Sort

This determines how your contact names are sorted for display in the Contacts screen. If set, the names will be sorted by surname. If not set (default), they will be displayed by first name.

Location Display

Preferences to do with how the address location (latitude and longitude) is displayed.

LAT/LONG Display - this determines the format in which the address location is displayed in the app.

If set (default), the address location will be displayed as degrees, minutes and seconds.

If not set, the address is displayed in decimal format.

Navigation Mode

Some apps or websites have a navigation mode. Use this setting to use your location with navgation or directions.

By default, the address location will be used in the standard way.

For example, in Google Maps, the location will just be plotted in the map.

If set, the location will be used as the destination for navigation or directions.

For example, in Google Maps, the location will be used as the destination for directions, showing the route.
