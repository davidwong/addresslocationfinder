Help -> Map

Map Screen

This is the map built into the app.

Hint: when you first locate an address on the map, the address location has a marker and is centred in the map.

Map Buttons

Centre the address location - if you move the map away from the address location, clicking this will centre it on the map again.

Centre your current location on the map.

Note: this button is disabled until your GPS can find your current location.

Show both locations (the address location and your current location) on the map. The map may have to be zoomed to fit both locations.

Note: this button is disabled until your GPS can find your current location.

Other Map Options

Toggle between map view and satellite view.

Menu -> View -> Satellite Mode

Toggle showing the compass on the map.

Menu -> View -> Display Orientation Compass

Details of the Locations

You can find out more information about the address location or your current location by tapping on the map markers.

Tap on the address location marker to pop up a dialogbox to show more details.

You can then click the Use Location button to pass the location to some other apps or websites.

Tap on the current location marker to see more details about where you are.

You can then click the Get Address button to find the address of your current location.

Note: the current location address may only be approximate, depending on how recent and how accurate your current location fix was.

GPS

GPS is used to find your current location on the map.

If your GPS is disabled, then a dialog may popup asking you if you want to enable it in your system settings.

If you leave your GPS disabled then the app will not be able to show your current location, but you can still see and use the address location.

This is also true if your GPS is unable to find your location.

Find Another Address to Locate

The menu has an option to select another address to location.

Menu -> New Destination

These are the same methods of finding and address as are available in the initial address finder screen.

Enter Address - manually enter another address, more...

Use Contact Address - select another address from your contact list, more ...

Get Address from SMS - select another address from a SMS message, more ...
