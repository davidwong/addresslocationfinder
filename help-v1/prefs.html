Help -> App Preferences

Preferences Screen

The user configurable application preferences.

Note: the preferences available may vary, depending on the version you have installed.

1

Address Lookup

Preferences to do with finding or locating an address.

0

Show Map - if set (default), when the address is located then the app will go to the built-in map. If not set, then the app will instead go to the Use Location screen.

Hint: this preference can also be set with the radiobuttons in the initial address finder screen.

Pick Location - when the geocoding service locates an address, sometimes it will return more than one possible location. If set, this preferences will pop up a dialog with a list of several of these locations to choose from. If not set (default), the app will simply use what the geocoding service thinks is the 'best' location.

Contact Name Display - This determines how your contact names are sorted for display in the Contacts screen. If set, the names will be sorted by surname. If not set (default), they will be displayed by first name.

Location Display

Preferences to do with how the address location (latitude and longitude) is displayed.

0

LAT/LONG Display - this determines the format in which the address location is displayed in the app.

If set (default), the address location will be displayed as degrees, minutes and seconds.

If not set, the address is displayed in decimal format.
